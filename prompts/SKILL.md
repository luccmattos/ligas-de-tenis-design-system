Read the `readme.md` file within this skill, and explore the other available files
(`styles.css` + `tokens/` for foundations, `guidelines/` for specimen cards,
`components/` for React primitives, `ui_kits/` for full screens, `assets/` for logos).

If creating visual artifacts (slides, mocks, throwaway prototypes, etc), copy assets out
and create static HTML files for the user to view — link `styles.css` for tokens and the
two signature textures (court-grid, diagonal hatch), and reuse the component patterns.
If working on production code, copy assets and read the rules here to become an expert in
designing with this brand.

Key brand rules to honor: Brazilian Portuguese, "você", warm-but-organized tone; Bebas Neue
display (ALL CAPS) + Outfit body; navy grounds / blue acts / green confirms / yellow accents
(50/25/12/8/5); soft corners, navy-tinted shadows, colored CTA glows; the tennis-ball 🎾 is
the only sanctioned emoji; logos are self-contained badges (light vs navy card variants).

If the user invokes this skill without other guidance, ask them what they want to build or
design, ask some questions, and act as an expert designer who outputs HTML artifacts _or_
production code, depending on the need.
