# Brand guidelines

## Value proposition

- **Core values:** Community, competition, flexibility, social connection, health
- **What we sell:** The experience of competing in a professional tennis circuit (ATP/WTA format) as an amateur

## Emotions to convey

| Emotion | How |
|---------|-----|
| Belonging | Real league, not weekend game |
| Energy | Match-day excitement |
| Trust | 10+ years, 1.000+ players, organized structure |
| Accessibility | Beginner to PRO; league meets you where you are |

## Visual differentiation

- Pro-sport aesthetic (ATP/WTA reference) in amateur context
- Premium, not club-newsletter

## What to avoid

- Purple gradients, generic tech startup aesthetics
- Playful/childish execution (emoji as primary UI, bouncy fonts)
- Washed-out muted palettes
- Stock photography that doesn't reflect the player community
- Generic sports design (neon-only palettes, busy layouts)

## Logo family

Logos are **self-contained badges** (silhouette + wordmark on rounded card), not bare marks.

| League | Code | Notes |
|--------|------|-------|
| Ligas de Tênis | `lt` | Umbrella / multi-league |
| Liga Alphaville de Tênis | `lat` | Flagship |
| LISP | `lisp` | São Paulo west/south |
| LIRJ | `lirj` | Rio de Janeiro |
| Liga Alphaville de Squash | `las` | Squash — not tennis |
| Ravenna Beach Tennis | `larb` | Beach tennis — **never** use tennis logos |

## Background textures

1. **Court-grid desk** — faint 24px navy grid on light gray; blue radial glow top-right
2. **Diagonal hatch** — ~−18° thin white lines at 6–8% opacity over navy
3. **Blue blur orbs** — depth on dark surfaces only; content surfaces stay opaque

## Iconography

Icon-light brand. No bundled icon font.

- Consumer/WhatsApp: tennis ball 🎾 only
- Status: solid color dots
- Logo silhouette is the primary iconographic device
- UI icons (if needed): Lucide outline 1.5–2px stroke; flag substitution
